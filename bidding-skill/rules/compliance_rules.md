# 合规红线与废标检查规则

## 废标检查清单

以下条目在投标文件生成完成后必须逐条核查，核查结果输出至 output/废标风险检查.md。

### 一、格式合规

- [ ] 投标文件是否按招标文件要求的组成结构编排
- [ ] 是否包含招标文件要求的全部必备章节
- [ ] 页码、目录是否与正文一致
- [ ] 法人代表签字页/授权书页是否保留占位
- [ ] 投标文件份数是否符合要求

### 二、实质性响应

- [ ] 全部带 ★ 的实质性技术参数是否逐条正面响应
- [ ] 是否存在实质性负偏离（可能导致废标）
- [ ] 技术方案是否覆盖了招标文件全部功能要求

### 三、报价合规

- [ ] 投标总价是否超过预算控制价/最高限价
- [ ] 分项报价之和是否等于投标总价（算术校验）
- [ ] 单价 × 数量是否等于分项合计
- [ ] 报价文件中是否有涂改或前后矛盾数据

### 四、资格合规

- [ ] 资质证书是否在投标截止日仍在有效期内
- [ ] 业绩证明是否满足招标要求的时间范围和数量
- [ ] 拟派人员是否满足招标要求的资质和数量

### 五、一致性校验

- [ ] 技术方案中的设备参数与报价文件中的品牌型号是否一致
- [ ] 商务方案中的交付周期与投标函中的承诺是否一致
- [ ] 偏离表中的声明与技术方案中的描述是否一致
- [ ] 不同章节中对同一事项的描述是否一致

### 六、串标嫌疑排查

- [ ] 文件中是否出现其他公司的名称、联系方式或项目信息
- [ ] 文件属性中是否残留其他公司信息
- [ ] 是否存在与常见串标特征相符的表述
