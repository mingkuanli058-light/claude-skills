# 废标条款

> 提取自：specs/招标文件.txt 第二部分"供应商须知资料表"、第三部分"供应商须知"及第七部分"评分标准"符合性审查
> 仅做信息提取，未做总结、改写、解释。原文如有"无效""拒绝"字样均逐条提取。

## 一、应答直接拒绝/无效情形（第二部分原文）

- 未提交应答文件正本的，其应答将被拒绝或作无效应答处理。

## 二、符合性审查不通过情形（第七部分原文）

| 序号 | 审查内容 |
|------|---------|
| 1 | 供应商的响应文件没有出现未按照遴选文件规定要求签署、盖章的情形 |
| 2 | 供应商的响应文件没有因为内容不全或字迹模糊辨认不清，导致响应文件不满足遴选文件实质性要求 |
| 3 | 供应商的响应报价没有超过遴选文件中规定的预算金额或者最高限价 |
| 4 | 供应商的响应文件没有出现属于遴选文件中所列无效响应情形 |
| 5 | 供应商的响应文件中列明的响应有效期≥90日历日 |
| 6 | 供应商的响应文件按照遴选文件的要求提供了供应商基本情况表 |
| 7 | 供应商的响应没有出现法律、法规和遴选文件规定的其他无效情形 |

> 注：上述符合性审查项目，任一项不满足则应答文件作无效处理。

## 三、遴选过程中影响公正导致无效情形（第三部分第21.1条原文）

在遴选期过程中，供应商企图影响采购人或遴选小组公正性的任何活动，将导致应答无效，并由其承担相应的法律责任。

## 四、串通应答无效情形（第三部分第21.2条原文）

有下列情形之一的，视为供应商串通，其应答无效：

1. 不同供应商的应答文件由同一单位或者个人编制；
2. 不同供应商委托同一单位或者个人办理应答事宜；
3. 不同供应商的应答文件载明的项目管理成员或者联系人员为同一人；
4. 不同供应商的应答文件异常一致；
5. 不同供应商的应答文件相互混装；

## 五、资格性审查不通过（第七部分原文，违反任一条则废标）

| 序号 | 审查内容 |
|------|---------|
| 1 | 供应商的响应文件按照遴选文件的要求提供了法人或者其他组织的营业执照等证明文件、自然人的身份证明 |
| 2 | 供应商的响应文件按照遴选文件的要求提供了法定代表人的授权委托书 |
| 3 | 供应商的响应文件按照遴选文件的要求提供了社会保障资金缴纳记录证明文件 |
| 4 | 供应商的响应文件按照遴选文件的要求提供了依法缴纳税收记录证明文件 |
| 5 | 供应商的响应文件按照遴选文件的要求提供了参加本次政府采购活动前三年内，在经营活动中没有重大违法记录的书面声明 |
| 9 | 信用信息查询结果显示供应商未被列入失信被执行人、重大税收违法案件当事人名单、政府采购严重违法失信行为记录名单 |
| 11 | 供应商的响应文件未出现内容不全或字迹模糊辨认不清，导致响应文件不满足遴选文件资格要求 |

## 六、遴选条件中的其他无效/废止情形

- **遴选文件领取要求**：供应商须按照遴选文件要求，在规定的领取时间内到采购人指定地点领取本项目遴选文件，否则应答无效。
- **采购活动终止情形之一**：报价超过采购预算的（采购活动终止）。
- 供应商不得被列入相关部门规定的失信被执行人、重大税收违法案件当事人名单、政府采购严重违法失信行为记录名单。（违反则不符合合格供应商条件）
