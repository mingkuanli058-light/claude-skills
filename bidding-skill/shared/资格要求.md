## 资格性审查（逐条原文）

| 序号 | 审查内容 |
|------|---------|
| 1 | 供应商的响应文件按照遴选文件的要求提供了法人或者其他组织的营业执照等证明文件、自然人的身份证明 |
| 2 | 供应商的响应文件按照遴选文件的要求提供了法定代表人的授权委托书 |
| 3 | 供应商的响应文件按照遴选文件的要求提供了社会保障资金缴纳记录证明文件 |
| 4 | 供应商的响应文件按照遴选文件的要求提供了依法缴纳税收记录证明文件 |
| 5 | 供应商的响应文件按照遴选文件的要求提供了参加本次政府采购活动前三年内，在经营活动中没有重大违法记录的书面声明 |
| 6 | 供应商的响应文件按照遴选文件的要求提供了响应保证金汇款凭证盖章复印件/保函等非现金形式盖章复印件（格式） |
| 7 | 供应商的响应文件按照遴选文件的要求提供了成交服务费承诺书（格式） |
| 8 | 供应商的响应文件按照遴选文件的要求提供了未违反《中华人民共和国政府采购法实施条例》第十八条规定的承诺（格式） |
| 9 | 信用信息查询结果显示供应商未被列入失信被执行人、重大税收违法案件当事人名单、政府采购严重违法失信行为记录名单 |
| 10 | 供应商的响应文件按照遴选文件的要求提供了其他资格/资质标准要求证明文件（如有） |
| 11 | 供应商的响应文件未出现内容不全或字迹模糊辨认不清，导致响应文件不满足遴选文件资格要求 |

## 合格供应商的基本条件（第三部分原文）

- 在中华人民共和国境内注册。
- 供应商应遵守中华人民共和国有关的国家法律、法规和条例。
- 供应商须按照遴选文件要求，在规定的领取时间内到采购人指定地点领取本项目遴选文件，否则应答无效。
- 供应商不得被列入相关部门规定的失信被执行人、重大税收违法案件当事人名单、政府采购严重违法失信行为记录名单。

## 供应商能力要求（第四部分原文）

1. 供应商应具备软件开发能力、安全服务能力及系统运行维护能力。
2. 具有丰富的相关项目服务经验，可根据采购人要求，提供包括但不限于项目应用系统的运维巡检、勤务值班、故障排除、内容维护、安全服务、数据及安防接口维护服务及其他服务。
3. 供应商应具备丰富的类似项目业绩，可在响应文件中提供合同复印件，合同至少提供合同首页、主要内容页、签字盖章页。
4. 供应商应结合项目具体情况，对项目有合理透彻的理解和分析，针对本项目关键点及重难点有一定的认识，并有相应的系统运维维护方案。供应商应根据采购文件的要求提供完整的值班维护方案、日常巡检方案、故障排除服务方案、内容维护及功能升级方案、安全服务方案、系统数据对接服务方案、服务响应及服务质量保证措施等。

## 联合体限制

- **是否允许联合体遴选**：否
