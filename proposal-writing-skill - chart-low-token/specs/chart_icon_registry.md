# Chart Icon Registry

本文件定义图件节点图标类型。

所有节点 type 必须从本文件选择。

---

# 一、服务器类

| type | 图标 |
|---|---|
| server | 服务器 |
| application_server | 应用服务器 |
| database | 数据库服务器 |

---

# 二、存储类

| type | 图标 |
|---|---|
| storage | 存储阵列 |
| nas | NAS存储 |

---

# 三、网络设备

| type | 图标 |
|---|---|
| core_switch | 核心交换机 |
| access_switch | 接入交换机 |
| firewall | 防火墙 |

---

# 四、终端设备

| type | 图标 |
|---|---|
| terminal | 业务终端 |
| camera | 摄像机 |
| workstation | 工作站 |

---

# 五、平台系统

| type | 图标 |
|---|---|
| platform | 平台系统 |
| application | 应用模块 |

---

# 六、禁止行为

禁止生成未注册节点类型。

如需新增节点类型：

必须修改本文件。
