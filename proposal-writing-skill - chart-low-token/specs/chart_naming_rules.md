# Chart Naming Rules

本文件定义图件节点命名规则。

用于保证图件表达统一。

---

# 一、命名原则

节点名称必须：

- 简短
- 明确
- 不包含品牌
- 不包含型号

---

# 二、服务器命名

正确示例：

应用服务器  
数据库服务器  

禁止：

华为服务器  
鲲鹏服务器  

---

# 三、存储命名

正确：

网络存储  
存储阵列  

禁止：

48盘位存储  

---

# 四、网络设备

正确：

核心交换机  
接入交换机  

禁止：

华三交换机  

---

# 五、终端设备

正确：

业务终端  
摄像机  

禁止：

某型号摄像机  

---

# 六、系统模块

正确：

执法视音频平台  
数据管理模块  

禁止：

系统V2.0  

---

# 七、命名统一原则

同一类型节点必须使用统一名称。

禁止：

应用服务器 / 业务服务器 混用。
