# 空间与布点参数

1️⃣ 标准型接报案室

适用范围：10间
尺寸：

长：6m

宽：4m

层高：3m

布点规则：

每间2台摄像机

覆盖接报案台

报案台里侧为民警，外侧为当事人

覆盖当事人区域

摄像机距离墙面300mm

拾音器位于接报案台上固定

输出：
生成1张标准布点图，其余同类空间按此执行。

2️⃣ 特殊型接报案室
适用范围：1间
适用范围：如有尺寸差异的房间
尺寸：

长：5 m

宽：4 m

差异说明：

面积变化

布点原则：
在标准规则基础上调整视角覆盖。

输出：
单独生成特殊型布点图。

五、说明

若11间结构完全一致，仅生成标准型布点图1张即可。
